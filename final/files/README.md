Retro Rewind
Retro Rewind is an arcade-inspired website where users can play retro-style games directly in their browser. The site is built for responsiveness and performance, working smoothly on both desktop and mobile devices. Featuring a neon aesthetic and a nostalgic vibe, this project brings classic gaming to the modern web.

Features:
Responsive Design - Works across desktop and mobile
Game Library - Includes a wide selection of open-source games
Fast Loading - Self-hosted fonts and preloading techniques for better performance
Neon Theme - CSS-based glowing effects and retro styling
Organized Navigation - Dropdown menu and genre categorisation

Tech Stack:
HTML5 - Semantic structure and content
CSS3 - Styling, edffects and responsive layouts
JavaScript - Interactive features and functionality
GitHub Pages - Hosting and deployment
Fonts - Self-hosted Press Start 2P and Roboto Mono
